Правила

Дайте публикации понятный заголовок

Выберите категорию поста

Добавьте изображение по нажатию кнопки «Выбирете файл»

Заполните аннотацию (краткое представленеи статьи), используя инструменты для форматирования текста

Заполните тело публикации, используя инструменты для форматирования текста

С помощью кнопки «Сохранить» выберите вариант сохранения:

- если всё в порядке, нажмите кнопку «Готово к публиуации»

- если еще нужно подумать нажмите кнопку «Сохранить в черновик» (Черновики видны только вам)

При нажатии кнопки «Отмена» ваши заполненые данные будут удалены
